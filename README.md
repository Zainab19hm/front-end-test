# Syrian Private University (SPU) - Web Portal Prototype

هذا المشروع عبارة عن نموذج أولي (Prototype) للواجهة الأمامية الخاصة بموقع الجامعة السورية الخاصة، تم بناؤه باستخدام تقنيات حديثة لضمان السرعة والأداء.

## 🛠 التقنيات المستخدمة
- **Vite**: كأداة بناء وتطوير سريعة.
- **Alpine.js**: لإدارة الحالة (State Management) والعمليات الديناميكية مثل السلايدر وتبديل اللغة.
- **Tailwind CSS v4**: للتنسيق البصري بناءً على الهوية البصرية للجامعة.
